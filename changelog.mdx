## Upgrading to the Latest Version

Follow these steps to update your Klutr instance safely.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your project files and database.

  </Step>
  <Step title="Update Package" icon="package">

    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn">
      ```bash
      npm install klutr@latest
      ```
      ```bash
      yarn add klutr@latest
      ```
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute database migrations if applicable.

    ```bash
    npx klutr migrate
    ```

  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your Klutr server and verify everything works.

  </Step>
</Steps>

<Callout kind="tip">
  Always test updates in a staging environment first. Check release notes for any breaking changes.
</Callout>
