## Overview

Klutr helps you create, organize, and maintain comprehensive documentation for your projects. You can build structured knowledge bases, collaborate with your team in real-time, and ensure your docs stay up-to-date with version control and search features. Whether you're documenting APIs, user guides, or internal wikis, Klutr streamlines the process so you focus on content rather than tools.

## Who Should Use Klutr

You should use Klutr if you're a developer needing API references, a product manager creating user guides, or a team lead organizing onboarding materials. Engineering teams benefit from its integration with Git workflows, while marketing and support teams appreciate the embeddable components and public sharing options. Small startups to large enterprises find value in its scalable structure.

## Quick Start

Get up and running in minutes.

<Steps>

<Step title="Create Account" icon="user-plus">

Sign up at `https://klutr.com/signup` using your email or GitHub account.

</Step>

<Step title="Set Up Space" icon="folder-plus">

Create a new documentation space for your project and invite collaborators.

````bash
# Example: Via CLI (optional)
npx klutr-cli init my-project-docs
````

</Step>

<Step title="Add Your First Page" icon="plus">

Use the editor to create markdown pages with live previews.

<CodeGroup tabs="Markdown,Embed">

```markdown
# Welcome to My Docs

This is your first page.
```

```html
<iframe src="https://klutr.com/embed/YOUR_SPACE_ID" width="100%" height="400"></iframe>
```

</CodeGroup>

</Step>

<Step title="Publish and Share" icon="globe">

Make your docs public or share private links with your team.

</Step>

</Steps>
